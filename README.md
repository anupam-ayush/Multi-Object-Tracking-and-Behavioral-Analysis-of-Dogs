# Multi-Object Tracking and Behavioral Analysis of Dogs

## Overview
This project implements a "Tracking-by-Detection" pipeline to detect, track, and analyze the behavior of multiple dogs and humans in dynamic video footage. Moving beyond simple bounding box detection, this system assigns unique identities to targets, mitigates occlusion issues, and categorizes real-time behavioral states (e.g., stationary, moving toward the user, interacting). 

## Key Features
* **Custom Object Detection:** Fine-tuned YOLOv8s model trained on the Roboflow "Stray Dogs" dataset.
* **Robust Multi-Object Tracking:** DeepSORT integration with Kalman filtering and deep appearance models for occlusion-resistant tracking and identity preservation.
* **Behavioral State Classification:** Rule-based algorithm utilizing centroid history to classify behaviors such as:
  * `Stationary`
  * `Moving Toward User`
  * `Moving Elsewhere`
  * `Interacting`
* **Trajectory Prediction & Visualization:** Calculates and visualizes past motion trails (historical centroids) and short-term future paths.
* **Annotated Video Output:** Generates rich output videos featuring bounding boxes, tracking IDs, behavioral states, confidence scores, and a real-time dashboard statistics overlay.

## Tech Stack
* **Language:** Python 3.8+
* **Computer Vision & Deep Learning:** Ultralytics (YOLOv8), DeepSORT (`deep-sort-realtime`), OpenCV (`opencv-python`), NumPy
* **Data & Environment:** Google Colab, Roboflow

## System Architecture
The pipeline consists of three primary phases:

### Phase I: Object Detection (YOLOv8)
Frames are processed to detect dogs and persons. Custom label mapping ensures all dog variants (e.g., with/without leash) are unified into a single tracking class. Duplicate or overlapping boxes are resolved using Non-Maximum Suppression (NMS).

### Phase II: Multi-Object Tracking (DeepSORT)
Detections are fed into the DeepSORT tracker. A Kalman Filter estimates the object's position in the next frame. The Hungarian algorithm matches tracks with new detections based on Intersection over Union (IoU) and a feature vector generated by a deep appearance model, ensuring robust re-identification even after occlusions.

### Phase III: Behavioral Analysis & Prediction
* **Stationary:** Triggers if the average Euclidean distance of recent historical centroids falls below a specific threshold (ε).
* **Moving Toward User:** Triggers if the distance between the dog's centroid and the user's centroid consistently decreases over a set window of frames.
* **Interacting:** Triggers if the bounding boxes of multiple dogs fall within a 50-pixel proximity threshold.

## Dataset
* **Source:** "Stray Dogs" Dataset via Roboflow.
* **Size:** 4,400+ images with ~10,000 annotations.
* **Preprocessing:** Applied data cleaning to merge disparate dog classes (0, 2, 3, 4, 5, 6) into a unified "Dog" category to improve tracker consistency. Augmented via Roboflow to improve model robustness.
